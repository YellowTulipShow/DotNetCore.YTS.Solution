# .Net Core 3.1 + 代码库

## 脚本

* 查找发布程序:

```powershell
./shell/find_all_project.ps1
```

* 发布程序:

```powershell
./shell/release.ps1
```

* 推送包到 Nuget 发布:

```powershell
./shell/push_nuget_packages.ps1
```

* 发布命令程序:

```powershell
./shell/release_command.ps1
```

* 安装已经发布的命令程序:

```powershell
./shell/install_command_packages.ps1
```
